Keys: 
- pad: select key
- A: press key
- B: is backspace
- L1: is shift
- R1: toggles key (useful for shift/ctrl...)
- L1: move cursor left
- R1: move cursor right
- Y: change keyboard location (top/bottom)
- X: show / hide keyboard
- Start: Enter
- Select: 
- MENU: quit
